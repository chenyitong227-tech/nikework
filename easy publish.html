把本地 HTML 变成一个可分享的链接

只需 5 步，把你写好的 HTML 文件上传到 GitHub，就能生成一个网页链接。别人可以在微信、飞书、浏览器直接打开 —— 不用装任何东西。

1

注册 GitHub 账号

全球最大的代码托管平台，注册免费
  1. 打开官网：https://github.com
  2. 点击右上角 Sign up，按提示输入邮箱、密码、用户名
  3. 去邮箱收激活验证码，完成注册
💡 用户名很重要：以后所有链接里都会包含它，建议起一个简洁、英文、无特殊符号的名字，比如 zhangsan-tech。
2

新建一个仓库（Repository）

仓库 = 存放你 HTML 文件的一个"网络文件夹"
  1. 登录后，点击右上角 + → 选择 New repository
  2. 填写仓库名，比如 nikework（英文、小写、不加空格）
  3. 勾选 Public（公开，别人才能通过链接访问）
  4. 勾选 Add a README file（可选，方便后续管理）
  5. 点击 Create repository 完成创建
⚠️ 一定要选 Public：Private（私有）仓库虽然可以创建，但需要付费才能开启 GitHub Pages，公开分享给同事和微信打开需要 Public。
3

上传本地 HTML 文件

直接把电脑上的文件拖进浏览器就行
  1. 进入刚才创建的仓库
  2. 点击页面中间的 Add file → 选择 Upload files
  3. 把你本地的 index.html（或其他 HTML 文件）拖拽到上传区域，或点击 choose your files 选择
  4. 如果有配套的 images/、css/、js/ 文件夹，一起拖进来
  5. 滚到页面底部，点绿色按钮 Commit changes
✅ 命名建议：把主文件命名为 index.html，这样最终链接可以省略文件名，直接是网站首页。
📌 文件名不要用中文：中文文件名在链接里会变成一串乱码（%E7%AE%80%E5%8D%95...），非常难分享。
推荐用英文小写 + 连字符：report.html、q3-sales.html。
4

开启 GitHub Pages（让文件变成网页）

这一步是"发布"，开启后才有对外访问的链接
  1. 在仓库页面顶部点击 Settings（齿轮图标）
  2. 在左侧菜单栏中找到 Pages
  3. 在 Build and deployment 区域：
    • Source 选择 Deploy from a branch
    • Branch 选择 main，右边文件夹选 / (root)
  4. 点击 Save 保存
  5. 等 1–2 分钟刷新页面，最上面就会出现绿色提示：
    "Your site is live at https://xxx.github.io/xxx/"
⏱️ 不用一直刷新：第一次发布需要等 1–2 分钟，之后每次更新 HTML 大概 20–60 秒生效。
5

拼接你的分享链接

按下面的公式就能自己拼出来，不用等系统提示

GitHub Pages 的链接格式非常固定，记住这个公式：

📐 链接公式

https://用户名.github.io/仓库名/文件名.html
用户名 = 注册时填的 username 仓库名 = 第 2 步创建的仓库 文件名 = 如果是 index.html 可省略

🌰 实际举例

用户名 仓库名 文件名 最终链接
chenyitong227-tech nikework index.html https://chenyitong227-tech.github.io/nikework/
chenyitong227-tech nikework report.html https://chenyitong227-tech.github.io/nikework/report.html
🎉 大功告成！把这个链接复制到微信、飞书、邮件里，任何人点开都能看到你的网页。

常见问题

同事最常问的几个坑
打开链接显示 404 怎么办？

1) 确认 Pages 已经开启（Settings → Pages 有绿色 "Your site is live"）；2) 确认文件名是英文且大小写正确；3) 第一次发布再等 1–2 分钟；4) 检查链接里的用户名、仓库名有没有拼错。

更新 HTML 后链接内容没变？

重新上传文件后需要 20–60 秒才会生效。同时浏览器有缓存，可以按 Cmd + Shift + R（Mac）或 Ctrl + F5（Win）强制刷新。

HTML 里的图片、CSS、JS 显示不出来？

把用到的资源文件（图片、样式、脚本）也一起上传到仓库，并且 HTML 里引用它们时用相对路径，例如 ./images/logo.png，不要用本地绝对路径 /Users/xxx/Desktop/...。

我的内容是保密的，不想公开怎么办？

GitHub Pages 需要 Public 仓库才能免费使用。如果不能公开，可以考虑：① 使用公司内部的静态托管平台；② 升级 GitHub Pro（Private 仓库也能用 Pages）；③ 用其他工具比如 Vercel、Netlify。

微信里点链接打不开？

少数情况下微信会拦截 github.io 域名，让对方点右上角"…"→"在浏览器打开"即可。或者可以把链接生成二维码分享。
